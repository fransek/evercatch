    Function createErr

    • createErr<E extends string, S = Error>(label: E, source?: S): Err<E, S>

      Creates an Err object.

      Type Parameters

      • E extends string

        A string literal type for the error label.

      • S = Error

        The type of the source error.

      Parameters

      • label: E

        The label for the error.

      • Optionalsource: S

        The source of the error. If not provided, a new Error with the label as the message will be created.

      Returns Err<E, S>

      An Err object.

      Example

      // Create an error with a source error object
      const cause = new TypeError("Something went wrong");
      const unexpectedError = createErr("UNEXPECTED_ERROR", cause);
      // unexpectedError is { label: "UNEXPECTED_ERROR", source: cause }

      // Create an error without a source
      const validationError = createErr("VALIDATION_ERROR");
      // validationError is { label: "VALIDATION_ERROR", source: Error("VALIDATION_ERROR") }
